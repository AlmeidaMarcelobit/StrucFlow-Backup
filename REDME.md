# Site de Inventário de Ativos Tecnológicos

Este é um projeto de site para gerenciamento de inventário de ativos tecnológicos. O site foi desenvolvido para ser simples, moderno e funcional, com uma estrutura de diretórios organizada e fácil de manter.

---

## 🚀 Como Executar o Projeto

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/seu-usuario/meu-site.git
   
   📂 Estrutura de Diretórios
    **CA estrutura do projeto está organizada da seguinte forma:**
    
        /meu-site/
        ├── /assets/                  # Pasta para todos os recursos estáticos
        │   ├── /images/              # Armazena fotos e imagens gerais
        │   ├── /icons/               # Ícones usados no site
        │   ├── /favicon/             # Favicon e suas variações
        │   ├── /css/                 # Arquivos de estilo (CSS)
        │   └── /scripts/             # Arquivos JavaScript
        ├── /pages/                   # Pasta para outras páginas do site
        ├── index.html                # Página inicial do site
        └── README.md                 # Documentação do projeto
        
        🛠 Tecnologias Utilizadas
            HTML: Estrutura do site.

            CSS: Estilização e design.

            JavaScript: Interatividade e funcionalidades dinâmicas.

            Favicon: Ícone da aba do navegador.
            
        🖼 Assets
            Imagens: Todas as imagens estão armazenadas na pasta /assets/images/.

            Ícones: Ícones utilizados no site estão na pasta /assets/icons/.

            Favicon: O favicon e suas variações estão na pasta /assets/favicon/.
            
        🎨 Personalização
            Para alterar o estilo do site, edite o arquivo CSS em /assets/css/styles.css.

            Para adicionar interatividade, edite o arquivo JavaScript em /assets/scripts/main.js.

            Para adicionar novas páginas, crie arquivos HTML na pasta /pages/ e adicione os links no menu de navegação.
            
        🤝 Como Contribuir
            Faça um fork do projeto.

            Crie uma branch para sua feature (git checkout -b feature/nova-feature).

            Commit suas mudanças (git commit -m 'Adicionando nova feature').

            Push para a branch (git push origin feature/nova-feature).

            Abra um Pull Request.
